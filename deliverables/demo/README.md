# Demo recording

1. Start `docker compose up --build`.
2. Capture: web login → dashboard → account → (optional) transaction; Grafana home; Postman/curl open banking with consent headers.
3. Export **`demo.mp4`** into this folder for the client USB / share.

**Git:** Add `demo.mp4` to `.gitignore` if files exceed repository limits; keep only this README in git.
