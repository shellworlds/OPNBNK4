# Web application guide

1. Open the portal URL (Compose: http://localhost:3000).
2. Sign in with **Keycloak** or use mock login if OIDC env vars are unset.
3. **Dashboard** lists accounts; a spinner shows while data loads.
4. Click an account for **transactions**; amounts over fraud review show status **UNDER_REVIEW** (highlighted).
5. Sign out from the header control.
